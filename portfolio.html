Portfólio

Aqui estão alguns dos projetos que desenvolvi ao longo da minha formação, com foco principal em Java, banco de dados, desenvolvimento web e construção de aplicações práticas.

Projeto em Destaque

  • Prioriza - Aplicação Full Stack
    Projeto de gerenciamento de atividades desenvolvido com Java, Spring Boot, PostgreSQL, React e TypeScript. No projeto, trabalhei com autenticação, operações de CRUD, validação, paginação, filtros, integração entre front-end e back-end e deploy da aplicação.

Outros Projetos

  • Locadora de Veículos - Banco de Dados
    Criação de banco de dados no MySQL com modelagem relacional, chaves estrangeiras, inserção de dados e consultas SQL com JOIN e GROUP BY.
  • Rede de Hotéis - Modelo Entidade-Relacionamento (MER)
    Elaboração de diagrama conceitual com entidades, relacionamentos, atributos e cardinalidades seguindo o padrão acadêmico.
  • Site Menstruação Sem Tabu
    Desenvolvimento de um site institucional com foco em apresentação de conteúdo, navegação clara e organização das informações do projeto.
  • Site Portfólio Pessoal
    Criação deste próprio site em HTML e CSS, com páginas separadas, navegação entre seções e apresentação das minhas informações acadêmicas e projetos.
